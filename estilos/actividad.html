¿Qué significa CSS?

significa CSS es una de las tres tecnologías que hacen la WEB. CSS significa Hojas de Estilo en Cascada en inglés Cascading Style Sheets y te dará los estilos (diseño) necesarios para que un sitio web o una aplicación web se vean visualmente bien.

Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc.

¿A qué se llama regla de CSS?

Las reglas son un conjunto de propiedades y valores que se aplicarán a una o varias etiquetas según necesitemos.

¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Selector:
selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos). Para dar estilo a un elemento diferente, solo cambia el selector.
Declaración
aplica las propiedades que querés dar al elemento seleccionado en el Selector. En este caso hay una sola regla, pero puede haber más.
Propiedades
es lo que te permite dar un estilo a un elemento HTML. En el ejemplo la propiedad es color que le dará color a las letras de los párrafos.
Valor de la propiedad
las propiedades deben tener un valor. Cada propiedad tiene valores definidos.

¿Qué es un selector en CSS?

un selector nos permitirá aplicar estilos a elementos específicos de HTML. De esta forma es posible ser preciso al momento de aplicar diseño a un sitio web.

Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

selector ID
el atributo ID de HTML identifica a un elemento HTML por medio de un nombre.
selector Clases
Los selectores de clases comienzan siempre por un . (punto).

Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML
  2. Convierte el HTML en un DOM
  3. el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los videos incrustados
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector.
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas
  6. En la pantalla se muestra el aspecto visual de la página (esta etapa se llama pintura).

Cuáles son las tres formas para vincular CSS con HTML

  1. utilizando la etiqueta STYLE
  2. utilizando el atributo STYLE
  3. utilizando archivo externo
